# FastApi TelegramBot WEBHOOK

1. install requirments python>=3.10 version:

    ```sh
    $ pip install -r requirements.txt
    ```


2. set webhook for bot:

    ```sh
    NGROK_URL = '' Replace with your webhook URL
    ```
    NGROK_URL

    ```sh
    $ python set_webhook.py 
    ```

3. run app

    ```sh
    $ uvicorn app.main:app
    ```
